# Análise de Risco de Crédito - Banco da Vida Imaginária

Este projeto consiste numa análise completa de Ciência de Dados para identificar padrões de incumprimento e automatizar a concessão de crédito através de modelos de Machine Learning.

## Contexto de Negócio
O Banco da Vida Imaginária é uma instituição financeira que procura reduzir perdas operacionais através da otimização da avaliação de risco dos seus clientes. O foco principal reside na segmentação de perfis e na predição da probabilidade de inadimplência.

## Objetivos do Projeto
- Analisar a taxa de inadimplência geral da instituição.
- Identificar variáveis demográficas e financeiras correlacionadas ao risco.
- Desenvolver um modelo preditivo baseado no algoritmo Random Forest.
- Fornecer suporte analítico para a tomada de decisão estratégica.

## Tecnologias Utilizadas
- Linguagem: Python 3.x
- Manipulação de Dados: Pandas e NumPy
- Visualização de Dados: Seaborn e Matplotlib
- Machine Learning: Scikit-Learn

## Estrutura do Repositório
- `credito_banco_vida_imaginaria.csv`: Base de dados com o histórico de 2.000 clientes.
- `analise_credito.ipynb`: Notebook contendo a Exploração de Dados (EDA) e a modelagem.
- `requirements.txt`: Ficheiro de configuração das dependências do sistema.
